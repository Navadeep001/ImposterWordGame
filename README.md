# Imposter Word Game

A browser-based multiplayer party game where one hidden player is the **Imposter** and everyone else knows the secret word.

Players reveal their roles one by one, discuss clues together, and try to identify who among them never saw the real word.

This project was built using **HTML, CSS, and JavaScript** with smooth UI transitions and interactive gameplay logic.


## How the Game Works

1. Select number of players (2–10)
2. Choose hint difficulty level
3. Each player privately reveals their role
4. Normal players see the secret word
5. The Imposter only sees a vague hint
6. Players discuss together
7. Vote to identify the Imposter
8. Reveal the result using the **Reveal Imposter** button


##  Features

- Interactive multi-player role reveal system
- Random imposter selection each round
- Difficulty-based hint system (Easy / Medium / Hard)
- Smooth flip-card role animation
- Tap-to-reveal gameplay experience
- Discussion phase support
- Reveal Imposter button at the end
- Replay game instantly with Play Again
- Clean modern UI with animated background particles


##  Technologies Used
